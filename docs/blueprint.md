# **App Name**: PolicyPal

## Core Features:

- User Authentication: Secure user authentication via Firebase to ensure policy queries are associated with unique users.
- Intelligent Policy Area Detection: Automatically identifies the relevant policy area (IT, HR, general) based on the user's question.
- Consolidated Answer Generation: The LLM combines retrieved information and reasoning to return a clear, concise, and complete answer to the user's policy question; incorporates tools for more accuracy.
- User Query Interface: A clean, intuitive interface where users can input their policy questions.
- Answer Display: Display the policy information and final answer to the user in an easily readable format.
- Query History: Stores previous queries associated with each unique user.
- Access History: Review recent search terms in chronological order.

## Style Guidelines:

- Primary color: Deep sky blue (#3498DB) for clarity and trust.
- Background color: Light gray (#ECF0F1) to provide a clean, neutral backdrop.
- Accent color: Emerald green (#2ECC71) for positive feedback and action buttons.
- Body and headline font: 'PT Sans' for a modern and warm user experience.
- Simple, recognizable icons representing IT, HR, and other policy categories.
- Clean and straightforward layout with clear sections for question input, search results, and query history.
- Subtle transitions and animations to acknowledge user actions.
